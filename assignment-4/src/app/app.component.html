2. The GameControl component should have buttons to start and stop the game

3. When starting the game, an event(holding a incrementing number) should get emitted each second (ref = setInterval())

4. The event should be listenable from outside the component

5. When stopping the game, no more events should get emitted(clearinterval(ref))

6. A new odd component should het created for every odd number emitted, the same should happen for the Even component(or even number)

7. Simply output odd-number or even-number in two components

Style the element (e.g: paragraph) holding your output text differently in both components
